## Run Locally

Clone the project

```bash
  git clone [(https://github.com/Gduro/SnakeGameJS.git)]
```

Go to the project directory

```bash
  cd SnakeGameJS
```
Start HTML file

```bash
  index.html
```


# How to play
Control your snake with arrows on keyboard.

You can play this game endless, so you can't win ;)

### Destiny of the game:

Get as many apples as possible

#### Good luck :)
